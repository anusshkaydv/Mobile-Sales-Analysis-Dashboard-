# 📊 Mobile Sales Analysis Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing mobile sales data using Power BI to generate meaningful business insights. The dashboard is designed to help understand sales performance, customer behavior, and transaction trends.

---

## 📁 Dataset Information
The dataset contains **3,835 records** with the following fields:

- Transaction ID  
- Day, Month, Year  
- Day Name  
- Brand  
- Units Sold  
- Price Per Unit  
- Customer Name  
- Customer Age  
- City  
- Payment Method  
- Customer Ratings  
- Mobile Model  

---

## 🧹 Data Cleaning & Preparation
- Handled missing values and ensured consistency  
- Converted data types (date, numeric fields)  
- Created derived columns (time-based analysis, categories)  
- Structured dataset for analysis  

---

## 📊 Dashboard Features

- 🔹 KPI Cards: Total Sales, Quantity, Transactions, Average  
- 🔹 Interactive Slicers: Month, Brand, Payment Method, Mobile Model  
- 🔹 Sales Analysis by City (Map Visualization)  
- 🔹 Monthly Sales Trend Analysis  
- 🔹 Customer Ratings Distribution  
- 🔹 Payment Method Analysis  
- 🔹 Mobile Model Performance Comparison  

---

## 📈 Key Insights

- Identified top-performing brands and mobile models  
- Observed monthly trends in sales and transactions  
- Analyzed customer satisfaction using ratings  
- Compared performance across cities and payment methods
